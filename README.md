# Object-Oriented Programming Through C++ - Lab Portfolio

This repository contains my laboratory assignments, programs, and problem-solving exercises completed during the 4th semester of my B.Tech program, focusing on core object-oriented concepts.

## Concepts Covered
* **Procedural Programming:** Functions, arrays, structures, and pointers in C++.
* **OOPs Fundamentals:** Classes, objects, memory allocation, and constructors/destructors.
* **Advanced OOPs:** Data abstraction, encapsulation, inheritance, polymorphism, and virtual functions.
* **File Handling & Templates:** Working with streams, files, and generic programming.

## How to Run the Code Locally
1. Clone this repository or download the `.cpp` files.
2. Open the file in an IDE like **Dev-C++**, **VS Code**, or **Code::Blocks**.
3. Compile using a C++ compiler (G++) and run the executable.
